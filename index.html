Code Examples

Real, copy-paste-ready code.

SmartLayout ‚Äî Animated breakpoint switching
SmartLayout(
  mobile: MobileNavigation(),
  tablet: TabletSidebar(),
  desktop: DesktopDrawer(),
  transitionDuration: Duration(milliseconds: 300),
  // Smooth crossfade when breakpoint changes!
)
SmartApp ‚Äî Drop-in MaterialApp replacement
void main() {
  runApp(SmartApp(
    title: 'My App',
    theme: ThemeData.light(),
    darkTheme: ThemeData.dark(),
    home: HomePage(),
    // Auto-configures breakpoints & responsive behavior
  ));
}
SmartForm ‚Äî Responsive form columns
SmartForm(
  // Auto: 1 col mobile, 2 tablet, 3 desktop
  children: [
    SmartFormField(child: TextField(decoration: InputDecoration(labelText: 'Name'))),
    SmartFormField(child: TextField(decoration: InputDecoration(labelText: 'Email'))),
    SmartFormField(span: 2, child: TextField(decoration: InputDecoration(labelText: 'Address'))),
  ],
)
Context Extensions ‚Äî Clean, readable responsive code
// Breakpoint checks
if (context.isMobile) showBottomSheet();
if (context.isDesktop) showSidePanel();

// Responsive values
final padding = context.responsive<double>(
  mobile: 16,
  tablet: 24,
  desktop: 32,
);

// Platform-adaptive
final style = context.adaptive(
  material: MaterialButton(),
  cupertino: CupertinoButton(),
);
SmartTheme ‚Äî Breakpoint-aware design tokens
SmartTheme(
  mobile: SmartThemeData(baseSpacing: 8, baseFontSize: 14),
  desktop: SmartThemeData(baseSpacing: 16, baseFontSize: 18),
  child: Builder(builder: (ctx) {
    final theme = SmartTheme.of(ctx);
    return Padding(
      padding: EdgeInsets.all(theme.baseSpacing),
      child: Text('Responsive!', style: TextStyle(fontSize: theme.baseFontSize)),
    );
  }),
)
